C'est quoi Dazure ?

Dzure est une application web développé en Python avec le framework Django qui sert à intéragir avec Le PaaS microsoft Azure

Comment je peux gérer mes ressources ?

Tout d'abord il faut avoir un compte microsoft azure et un compte Dazure, Sinon vous ne pouvez pas accéder à l'application.

Des exemples d'utilisation de l'application

  • Créer et gérer un groupe de ressources (Stockage,Réseau virtuelle,...)
  • Lancement d'une machine virtuelle
  • Lancement d'un script d'automatisation
